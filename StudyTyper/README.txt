### INSTALATION GUIDE ###

## Flask
1: Make sure you have Python installed

2: Install Python Extension to IDE

3: In IDE terminal, run "pip install flask"

## Ollama:
1: Download Ollama at https://ollama.com/download/windows

2: Run Ollama (Don't worry if nothing pops up, runs in background)

3: To make sure you have Ollama installed properly, run the command “ollama” in terminal

4: Install proper Ollama model by typing in terminal “ollama run llama3.2:1b” (For smallest model)

* You can start typing to chat with model. Type /bye to exit *

5: Run command in terminal of project “pip install ollama”
(Use “brew install ollama” on macos)

6: All good to run program now with Ollama AI working

### RUNNING GUIDE ###
1: Make sure you have all pre-reqs installed

2: Ollama MUST be running (click ^ in bottom right of your windows task bar to check)

3: Make sure current cirectory is /StudyTyper

4: In IDE terminal, run "python app.py"

5: In browser, go to http://127.0.0.1:5000 to view and use the app
